# 끄코 끝말잇기

**끄투코리아 단어 데이터 기반 터미널 끝말잇기 게임**
##### 이 저장소는 바이브코딩을 이용하여 제작되었습니다.

## 실행 방법

```bash
cd kkutu
pip install requests
python main.py
```

## 요구사항

- Python 3.9 이상
- requests 라이브러리

## 끝말잇기 규칙

**1. 상대방이 말한 단어의 마지막 글자로 시작하는 단어를 적어야합니다**
**2. 이미 사용한 단어는 다시 사용할 수 없습니다**
**3. words.json 안에 있는 단어들만 사용 가능합니다**
**4. 두음법칙이 적용됩니다**
**5. 이 끝말잇기는 소설 이름과 애니 이름, 게임 안에있는 맵 이름등 허용이 관대합니다**

## 게임 명령어

| 명령어 | 설명 |
|--------|------|
| `/hint` | 현재 글자로 시작하는 단어 힌트 |
| `/show` | 가능한 단어 목록 표시 |
| `/skip` | 이번 턴  포기 |
| `/end` | 게임 종료 후 결과 보기 |
| `/quit` | 게임 종료 |

## 난이도 설정

| 난이도 | 설정 |
|--------|------|
| `쉬움` | 2~3글자 단어 |
| `보통` | 3~5글자 단어 |
| `어려움` | 6~9글자 단어 |
| `극악` | 이을수있는 가장 최적의 단어 |
